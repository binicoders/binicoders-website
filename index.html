¿Qué es react-native?

Una tecnología de código abierto de Facebook que permite crear aplicaciones para Android y ios con una única base de código

Un único código

Se programan una vez con tecnología web y se generan aplicaciones nativas para Android y ios

Aplicación nativa, nativa

El resultado es una aplicación nativa de verdad. Ni híbrida ni webview

El doble de rápido

Sólo se programa una vez, por lo que los tiempos de entrega son el doble de rápidos

Coste más ajustado

Menos horas de programación equivalen a precios más ajustados. Así de simple.
